# Markdown Formatting
## Title:
``` markdown
# Title here
```
# Title here
## Header:
``` markdown
# Header here
```
## Header here
## New line:
``` markdown
end of line\
new line
```
end of line\
new line
## Dot-point list:
``` markdown
- Main point
    - Sub point
        - Another sub point
```
- Main point
    - Sub point
        - Another sub point
## Codeblock:
    ``` markdown
    insert code here
    ```
## Bold:
``` markdown

```
**bold text here**
## Italics:
``` markdown
*italics text here*
```
